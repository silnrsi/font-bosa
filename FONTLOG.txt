FONTLOG for the Lisu New fonts

This file provides detailed information on the Lisu New Font Software. This information should be distributed along with the Lisu New fonts and any derivative works.

Basic Font Information

Lisu New is a Unicode typeface family that supports all languages that use the Lisu script and its variants, and could be expanded to support other scripts.
